# droid-nix

Nix package for the [Factory CLI](https://factory.ai) (`droid`) — prebuilt binary,
patched for NixOS via `autoPatchelfHook`.

## Use

```nix
let
  droidSrc = builtins.fetchTarball {
    url = "https://github.com/farwydi/droid-nix/archive/refs/heads/master.tar.gz";
  };
  droid = (import <nixpkgs> {
    inherit (pkgs.stdenv.hostPlatform) system;
    config.allowUnfree = true;
  }).callPackage "${droidSrc}/package.nix" { };
in
{
  home.packages = [ droid ];
}
```

## Updating

`version` and per-system hashes live in `sources.json`, regenerated by `./update.sh`
(pulls the latest version from the official install script). CI runs it daily and
commits any bump. Pin a version: `./update.sh 0.156.2`.
